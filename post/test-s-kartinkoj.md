## Как добавлять картинки

Есть два способа:

### Способ 1: Внешние ссылки (как здесь)

Просто используй URL:

![Описание](https://picsum.photos/600/300)

### Способ 2: Локальные файлы

1. Загрузи картинку в папку `assets/images/` на GitHub
2. Укажи путь: `/codesound/assets/images/tvoja-kartinka.jpg`

Пример:
![Описание](/codesound/assets/images/dress-russ2.jpg)
## Форматирование

В Markdown можно использовать:

- **Жирный текст** с помощью `**текст**`
- *Курсив* с помощью `*текст*`
- [Ссылки](https://github.com)
- Код и многое другое!
